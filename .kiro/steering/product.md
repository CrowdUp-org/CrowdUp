# Product Overview

CrowdUp is a social feedback platform where users can submit and vote on bug reports, feature requests, and complaints about various companies and products. The platform features a Reddit-style voting system with a podium view for top posts, user profiles, company pages, and messaging functionality.

## Core Features

- Post creation for bug reports, feature requests, and complaints
- Upvote/downvote system for community feedback
- Podium view highlighting top 3 posts
- Company-specific pages
- User profiles and messaging
- Search and trending views
- Real-time notifications

## User Experience

The app uses a modern, clean design with a floating header, gradient accents (yellow-to-orange), and smooth transitions. The interface emphasizes community engagement and makes it easy to discover and interact with feedback posts.
