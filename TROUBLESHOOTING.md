# Troubleshooting Guide

## Common Issues & Solutions

This document provides solutions for common problems encountered while using CrowdUp.

---

## Authentication Issues

### Unable to Login

**Problem:** Users cannot log in with correct credentials.

**Causes:**
- Incorrect email or password
- Account not verified
- Account has been deactivated
- Session token has expired

**Solutions:**
1. Verify email and password are typed correctly (check for caps lock)
2. Check your email for verification link if account is new
3. Try password reset: Click "Forgot Password" on login page
4. Clear browser cookies and cache, then try again
5. Try logging in from a different browser or incognito mode
6. Contact support if issue persists

### "Invalid Token" Error

**Problem:** Receiving "Invalid Token" or "Unauthorized" errors.

**Causes:**
- Session expired
- Token corrupted
- Browser cache issues

**Solutions:**
1. Log out and log back in
2. Clear all browser cookies
3. Clear browser cache
4. Disable browser extensions temporarily
5. Try a different browser

### Password Reset Not Working

**Problem:** Password reset email not received.

**Causes:**
- Email in spam/junk folder
- Email address not registered
- Password reset link expired

**Solutions:**
1. Check spam and junk folders
2. Verify the email address registered with account
3. Request a new password reset link (previous one may have expired)
4. Whitelist notifications@crowdup.com in email settings
5. Contact support for manual password reset

## Account & Profile Issues

### Cannot Update Profile

**Problem:** Changes to profile don't save or show errors.

**Causes:**
- Network connectivity issues
- Browser cache not cleared
- Invalid input data
- Session expired

**Solutions:**
1. Check internet connection
2. Clear browser cache and cookies
3. Verify all required fields are filled correctly
4. Log out and log back in
5. Try a different browser
6. Check file size if uploading avatar (max 5MB)

### Avatar Upload Issues

**Problem:** Cannot upload profile picture.

**Causes:**
- File too large
- Unsupported file format
- Browser doesn't support file upload
- Server storage issues

**Solutions:**
1. Ensure image is in JPEG or PNG format
2. Compress image if larger than 5MB (use tools like TinyPNG)
3. Resize image to recommended size: 500x500px
4. Try using a different browser
5. Clear browser cache
6. Try on a different device

### Email Change Not Confirming

**Problem:** Cannot change email address or confirmation email not received.

**Causes:**
- Confirmation email in spam
- Email already in use
- Link expired

**Solutions:**
1. Check spam/junk email folder
2. Verify new email address is not already registered
3. Request new confirmation email if link expired
4. Whitelist notifications@crowdup.com
5. Wait 5 minutes before trying again

## Post & Content Issues

### Post Not Publishing

**Problem:** Post remains in draft or shows error when publishing.

**Causes:**
- Network connection lost
- Content too long
- Invalid characters in content
- Session expired

**Solutions:**
1. Check internet connection
2. Ensure post is under 10,000 characters
3. Remove special characters that might cause issues
4. Log out and back in
5. Try in a different browser
6. Save as draft and try publishing again later

### Cannot Edit or Delete Post

**Problem:** Edit/delete buttons not available or showing errors.

**Causes:**
- Not the post author
- Post has been archived
- Session expired
- Insufficient permissions

**Solutions:**
1. Verify you are the post author
2. Check if post is within edit window (usually 30 minutes)
3. Log out and log back in
4. Contact support if permissions issue

### Images Not Loading in Posts

**Problem:** Images show broken link icon or don't display.

**Causes:**
- Image link broken
- Image deleted from source
- Network connectivity issues
- Browser cache issue

**Solutions:**
1. Clear browser cache
2. Try in incognito/private mode
3. Check if image URL is still valid
4. Re-upload the image
5. Try different image format
6. Check internet connection

### Search Not Finding Posts

**Problem:** Search results are empty or incomplete.

**Causes:**
- Search index not updated
- Using very specific search terms
- Posts are private or restricted
- Network issue

**Solutions:**
1. Try broader search terms
2. Use filters to narrow results
3. Try searching by author name
4. Check if posts are set to private
5. Wait a few minutes for search index to update
6. Refresh the page

## Comment & Voting Issues

### Cannot Post Comments

**Problem:** Comment button disabled or submission fails.

**Causes:**
- Not logged in
- Insufficient permissions
- Post comments disabled
- Session expired

**Solutions:**
1. Ensure you are logged in
2. Refresh the page
3. Check if comments are enabled for this post
4. Log out and back in
5. Try in a different browser

### Votes Not Registering

**Problem:** Upvote/downvote not counting or reverting.

**Causes:**
- Network delay
- Session expired
- Server processing issue
- Double-submission

**Solutions:**
1. Wait 5 seconds and refresh page
2. Check your internet connection
3. Log out and back in
4. Clear browser cache
5. Try again in a few moments

## Performance Issues

### Website Loading Slowly

**Problem:** Pages take long time to load.

**Causes:**
- Slow internet connection
- Browser cache full
- Too many browser extensions
- Server under heavy load

**Solutions:**
1. Check internet connection speed
2. Clear browser cache and cookies
3. Disable browser extensions temporarily
4. Close unnecessary browser tabs
5. Try during off-peak hours
6. Update your browser to latest version

### "Server Error" Messages

**Problem:** Seeing 500 or 503 error messages.

**Causes:**
- Temporary server issues
- Maintenance in progress
- High traffic

**Solutions:**
1. Wait 5-10 minutes and try again
2. Check CrowdUp status page for updates
3. Try in a different browser
4. Check social media for maintenance announcements
5. Contact support if issue persists

### High CPU/Memory Usage

**Problem:** Website using excessive system resources.

**Causes:**
- Browser cache full
- Too many open tabs
- Outdated browser
- Extension issues

**Solutions:**
1. Close unnecessary tabs and windows
2. Clear browser cache
3. Update browser to latest version
4. Disable extensions
5. Restart browser
6. Try a different browser

---

## Mobile App Issues

### App Crashes on Launch

**Problem:** App closes immediately after opening.

**Causes:**
- Outdated app version
- Corrupted app data
- Insufficient device storage
- OS compatibility issue

**Solutions:**
1. Update app to latest version
2. Clear app cache (not data)
3. Free up device storage space
4. Restart your device
5. Uninstall and reinstall app
6. Check OS version compatibility

### Push Notifications Not Working

**Problem:** Not receiving notifications from app.

**Causes:**
- Notifications disabled
- App not authorized for notifications
- Device permission not granted

**Solutions:**
1. Enable notifications in app settings
2. Check device notification settings
3. Ensure app has notification permission
4. Reinstall app to re-request permissions
5. Restart device

## Getting Additional Help

### Support Channels

1. **Email Support**: support@crowdup.com
2. **Contact Form**: [Contact Page](/contact)
3. **Community Forum**: [Forum Link](/community)
4. **GitHub Issues**: [Report Bug](https://github.com/CrowdUp-org/CrowdUp/issues)

When reporting issues, please include:
- Browser and version
- Operating system
- Steps to reproduce
- Screenshots if applicable
- Error messages

### Creating a Support Ticket

1. Go to support page
2. Click "New Ticket"
3. Select issue category
4. Describe your problem in detail
5. Attach screenshots/logs if available
6. Submit and track with ticket ID
