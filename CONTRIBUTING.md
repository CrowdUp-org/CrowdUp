# Contributing to CrowdUp

Thank you for your interest in contributing to CrowdUp!

## Important: License Agreement

By submitting a pull request or issue, you agree to the terms of our [LICENSE](./LICENSE), including the automatic grant-back of rights to the project owner.

## How to Contribute

1. **Fork** the repository (only for preparing a PR)
2. **Create a branch** for your fix/feature
3. **Make your changes** with clear commit messages
4. **Submit a pull request** to the main repository
5. **Delete your fork** after the PR is merged/closed

## What We Accept

- Bug fixes
- Performance improvements
- Documentation improvements
- New features (please open an issue first to discuss)

## Code of Conduct

Be respectful and constructive. We reserve the right to reject any contribution.
